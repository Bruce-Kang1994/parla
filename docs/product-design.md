## 一、竞品全景分析

| **企业版** | 有(定制) | 无 | 无 | 有(HIPAA/SOC2) | 无 | 无 |

### 1.2 各竞品核心差异化

- **Typeoff**: 最便宜订阅 + 最大免费额度 + 开发者定位 + "忠实原话"轻编辑
- **Spokenly**: 本地模式完全免费无限制 + Agent Mode(语音控制Mac) + 10万+用户
- **Typeless**: 跨平台最全(macOS/Win/iOS) + 中国用户多
- **Wispr Flow**: Command Mode(语音编辑文本) + 企业合规(HIPAA/SOC2)
- **VoiceInk**: 开源 + 一次性买断 + Power Mode(按应用自动切换配置)
- **Superwhisper**: 最强离线 + 可定制AI模式/Prompt

### 1.3 Typeoff Onboarding 流程 (10步)

| 步骤 | 标题 | 内容 |
|------|------|------|
| 1 | 选择语言 | 下拉选择 UI 语言 |
| 2 | 隐私政策 | 3 条隐私承诺(自研引擎/不共享/本地存储) |
| 3 | 登录 | 浏览器 OAuth 登录/注册 |
| 4 | 设置权限 | 辅助功能 + 输入监控两项权限 |
| 5 | 测试麦克风 | 音频设备选择 + 音量指示器 |
| 6 | 测试快捷键 | 按 Fn 键验证键盘兼容 |
| 7 | 试用一下 | 引导式首次听写(朗读示例文字) |
| 8 | 60+语言 | 展示支持的语言列表 |
| 9 | 支持的应用 | 展示 25+ 兼容应用图标 |
| 10 | 一切就绪 | 撒花庆祝 + "开始使用"按钮 |

**设计规律**: 设置(1-3) → 权限(4) → 硬件验证(5-6) → 上手体验(7) → 价值展示(8-9) → 庆祝(10)

### 1.4 Typeoff 主页与设置

**主页:** 使用统计(听写时长/字数/速度/节省时间) + 使用教程(3步)
**侧边栏:** 首页 / 词汇 / 历史记录 / 设置 / 邀请好友($8) / 用户信息 / 关于
**设置面板:**
- 语音识别: 转录语言 / 输出语言 / 音频设备
- 交互: 录音快捷键(Fn) / 交互音效
- 外观: 语言 / 外观主题(系统/亮/暗)

### 1.5 Typeoff 增长机制

- **邀请奖励**: 推荐人获 $8 账户余额，被推荐人获 $10 余额(日限 5 次)
- **礼品卡**: GC-XXXXXXXXXXXX 格式，可兑换 1 年 Pro
- **社区推广**: V2EX 前 50 名用户赠 $120(1年Pro)，其他用户赠 $30(3个月Pro)

## 二、Murmur 产品定位

### 2.1 差异化策略

在六款竞品中找到 Murmur 的生存空间:

| 竞品策略 | Murmur 策略 |
|---------|-----------|
| Typeoff: 轻编辑保留原话 | **Murmur: 智能场景化输出** — 根据不同场景(邮件/代码/聊天)自动调整输出格式和语气 |
| Spokenly: 本地免费无限制 | **Murmur: 云端+本地混合** — 基础转录本地免费，AI润色云端付费 |
| VoiceInk: 开源一次性买断 | **Murmur: 免费额度慷慨** — 周 10,000 字免费(超过所有竞品) |
| Wispr Flow: 企业合规 | **Murmur: 个人开发者优先** — 与 IDE 深度集成，代码注释/commit message 场景优化 |

### 2.2 核心 Slogan

**中文**: 你说，我来写。
**英文**: You talk, I type.
**日文**: あなたが話す、私が書く。

### 2.3 目标用户

1. 中文开发者 (V2EX / 即刻 / 少数派社区)
2. 需要大量文字输出的知识工作者
3. 多语言使用者(中英日切换)

## 五、定价策略

### 5.1 定价对比与决策

| | Typeoff | Wispr | Typeless | **Murmur** |
|---|---------|-------|----------|-----------|
| 免费额度 | 8,000字/周 | 2,000字/周 | 4,000字/周 | **10,000字/周** |
| 月费 | $9.90 | $12 | $30 | **¥59 (~$8)** |
| 年费 | $99 | $144 | $144 | **¥499 (~$69)** |
| 年付折扣 | 17% | 0% | 60% | **30%** |

**策略: 最大免费额度 + 最低价格 → 快速获取中文用户市场**

### 5.2 套餐设计

**Free (免费)**
- 10,000 字/周
- 全部场景模式
- 基础 AI 润色
- 历史记录(最近 7 天)

**Pro (¥59/月 或 ¥499/年)**
- 无限字数
- 跨语言输入(说中文出英文)
- 自定义词典
- 优先处理速度
- 历史记录(无限)
- 邮件支持

**Team (未来)**
- 团队管理后台
- 统一计费
- 使用报告

### 5.3 增长机制

**邀请码 (Invite Code)**
- 格式: `MUR-INV-XXXXXX`
- 推荐人: 每成功邀请 1 人，获 5,000 字/周额外额度(累计上限 50,000字)
- 被推荐人: 首周 20,000 字(翻倍)
- 日限: 10 个邀请/天
- 触发条件: 被推荐人使用超过 1,000 字后双方才生效

**兑换码 (Redemption Code)**
- 格式: `MUR-PRO-XXXXXXXXXXXX`
- 可兑换: 1个月/3个月/6个月/1年 Pro
- 用途: 社区推广、KOL 合作、活动赠品
- 管理: Web 后台批量生成、查看使用状态

**社区推广计划**
- V2EX: 前 50 名用户赠 6 个月 Pro + 帖子分享
- 即刻: 前 100 名用户赠 3 个月 Pro
- 少数派: 评测合作，读者专属 1 个月 Pro

## 六、支付模块设计

### 6.1 支付方式

| 市场 | 支付方式 | 技术方案 |
|------|---------|---------|
| 海外 | 信用卡/Debit | Stripe Checkout |
| 中国 | 支付宝 | Stripe 支付宝集成 |
| 中国 | 微信支付 | Stripe 微信支付集成 |

> Stripe 同时支持信用卡、支付宝、微信支付，一个后端搞定。

### 6.2 支付流程

```
桌面客户端 [升级Pro]
       │
       ▼
打开浏览器 → Stripe Checkout 页面
       │
       ▼
选择月付/年付 → 支付(信用卡/支付宝/微信)
       │
       ▼
Stripe Webhook → 后端更新用户订阅状态
       │
       ▼
客户端轮询/WebSocket → 检测到 Pro 激活 → UI 更新
```

### 6.3 后端需求

需要一个轻量后端服务(可用 Vercel + Supabase):

| API | 功能 |
|-----|------|
| `POST /api/auth/login` | 邮箱验证码登录 |
| `GET /api/user/profile` | 获取用户信息+订阅状态+用量 |
| `POST /api/user/usage` | 上报用量(每次听写完成) |
| `POST /api/payment/checkout` | 创建 Stripe Checkout Session |
| `POST /api/payment/webhook` | Stripe Webhook 回调 |
| `POST /api/invite/create` | 生成邀请码 |
| `POST /api/invite/redeem` | 使用邀请码 |
| `POST /api/redeem` | 使用兑换码 |
| `GET /api/user/history` | 同步历史记录(可选) |

### 6.4 用户账户体系

MVP 阶段用最简方案:
- **邮箱 + 验证码登录**(无密码)
- 用户数据存 Supabase
- API Key 加密存储(AES-256)
- 订阅状态通过 JWT Token 下发给客户端
